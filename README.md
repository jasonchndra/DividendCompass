# Dividend Compass V1.0
Dividend Compass is a minimalist fintech utility designed to fill a critical gap in traditional brokerage apps like Stockbit, which lack integrated, lifestyle-oriented yield calculators. By automating ticker sanitization and the 10% final tax deduction, the tool allows investors to calculate their net dividends significantly faster than manual methods. Built with Python and a serverless architecture, it features a Milestone Engine that transforms raw IDX data into relatable insights—mapping passive income against real-world costs like specialty coffee or gym memberships. This project serves as a proof-of-concept for data-driven products that prioritize user experience, financial literacy, and rapid, actionable financial snapshots.
